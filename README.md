# Multi-Step-Form-UI
A clean and responsive multi-step form built using HTML, CSS, and JavaScript. This project demonstrates step-by-step form navigation with a modern user interface.

Live Demo -

🚀 Features
Step-by-step form flow (Account → Details → Payment → Review)
Smooth navigation between steps
Form validation (basic)
Progress indicator UI
Clean and minimal design
Responsive layout

🛠️ Tech Stack
HTML5
CSS3
JavaScript

📂 Project Structure
│── index.html
│── style.css
│── script.js

✨ Future Improvements
Add backend integration
Store form data
Add animations/transitions
Improve validation
Dark mode support

⚙️ How It Works
User starts at Account step
Clicks Next to move forward
Each step updates the UI and progress bar
Final step shows Review & Submit
